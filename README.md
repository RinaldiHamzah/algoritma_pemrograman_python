# Algoritma Pemrograman Python

Repository ini berisi kumpulan materi, contoh kode, dan latihan mengenai **algoritma dan pemrograman menggunakan Python**, yang dirancang untuk mendukung pembelajaran dasar pemrograman hingga penerapannya dalam **analisis data, machine learning, dan pengolahan teks**.

## Tujuan
Repository ini dibuat untuk:
- Memahami konsep dasar algoritma dan logika pemrograman
- Menguasai Python sebagai bahasa pemrograman utama
- Mengimplementasikan algoritma dalam konteks data nyata
- Mempersiapkan dasar untuk data science dan machine learning
